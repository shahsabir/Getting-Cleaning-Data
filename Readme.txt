This file describes how run_analysis.R script works.

1) unzip the data from https://d396qusza40orc.cloudfront.net/getdata%2Fprojectfiles%2FUCI%20HAR%20Dataset.zip 
and rename the folder with "train".
Make sure the folder "train" and the run_analysis.R script are both in the current working directory.
2) use source("run_analysis.R") command in RStudio.
3) you will find one output files Result.txt ( it contains a data frame called result with 180*68 dimension) generated in the current working directory
